Questão 5 - Recomendações do código

  • O texto não possui <header>, <main>. Que deixam o código com mais significado.

  • Texto da id=”principal” não está dentro de nenhuma tag, uma opção seria colocar com a tag <p>, um texto sem tag não possui valor semântico, impossibilitando a associação de significados e atributos;

  • O estilo CSS da id=”cabecalho” está inline, não é uma prática recomendada. O ideal é colocar em um arquivo separado;

  • A tag imagem também está com o CSS inline, o recomendado é arquivo separado;

  • A tag imagem está dentro de h1, o que é errado;

  • A tag <b> não é mais utilizada pois existem outras técnicas que foram implementadas que são mais recomendáveis. Como por exemplo <strong> ou utilizar a propriedade font-weight do CSS;
